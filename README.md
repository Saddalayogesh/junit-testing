


# JUnit Testing Project 🚀

## 📌 Overview

This project is a Java-based Customer Management System built to practice JUnit 5 testing, Maven project management, and core Java backend development concepts. It demonstrates CRUD operations, exception handling, and unit testing best practices.

---

## ✨ Features

* Add Customer
* View Customer by ID
* View All Customers
* Update Customer Details
* Delete Customer
* Custom Exception Handling
* Positive Test Cases
* Negative Test Cases
* Maven Build Setup

---

## 🛠 Tech Stack

* Java
* JUnit 5
* Maven
* Git
* GitHub

---

## 📂 Project Structure

src/main/java/com/java/testing/

├── model        Customer entity

├── service      Business logic

├── exception    Custom exceptions

src/test/java/com/java/testing/

└── service      JUnit test classes

---

## ▶️ How to Run

### 1. Clone Repository

git clone [https://github.com/Saddalayogesh/junit-testing.git](https://github.com/Saddalayogesh/junit-testing.git)

### 2. Build Project

mvn clean install

### 3. Run Tests

mvn test
